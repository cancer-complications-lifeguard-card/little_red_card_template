## 🎯 模板特点

- **呼吸评估**：呼吸频率、血氧饱和度、呼吸音评估
- **紧急处理**：吸氧、体位、药物使用指导
- **转诊指征**：明确的医院转诊标准
- **家庭护理**：日常呼吸康复和预防措施
