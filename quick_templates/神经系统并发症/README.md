## 🎯 模板特点

- **神经系统评估**：意识水平、瞳孔反应、肢体活动评估
- **早期识别**：颅内压增高、脑疝的危险信号
- **紧急处理**：体位管理、降颅压措施
- **转诊时机**：神经外科急诊的转诊指征
