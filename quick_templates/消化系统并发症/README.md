## 🎯 模板特点

- **标准化结构**：急救指导 → 病情诊断 → 日常预防 → 辅助服务
- **120报警话术**：专业的急救电话沟通模板
- **关键信息记录**：患者信息、症状记录、处理过程
- **交互功能**：可填写、保存、打印的HTML格式
