# 急症处理卡生成提示词模板

## 基础模板生成提示词

```
你是一位经验丰富的医疗专家，请为[并发症名称]创建一个急症处理指引卡片。

要求：
1. 按照四大模块结构：急救指导步骤 → 病情诊断关键指引 → 日常注意和预防 → 辅助服务
2. 包含120报警话术模板
3. 提供关键信息卡（患者基本信息、血型、联系方式等）
4. 包含风险评估体系
5. 提供专业医生推荐和医院选择策略
6. 包含日常预防和康复管理
7. 格式为HTML，包含CSS样式和JavaScript交互功能

请确保内容专业、实用、易懂，适合患者和家属使用。
```

## 知识库整合提示词

```
请基于以下知识库内容，为[并发症名称]生成更准确的处理指引：

知识库内容：
[插入相关知识库文档]

要求：
1. 整合专业医学知识和实际经验
2. 突出关键风险点和处理要点
3. 提供个性化建议
4. 确保信息的准确性和时效性
```

## RAG增强提示词

```
作为医疗AI助手，请基于检索到的相关文档，回答患者关于[并发症名称]的问题：

检索文档：
[RAG检索结果]

患者问题：
[用户输入]

回答要求：
1. 基于检索文档提供准确信息
2. 如果信息不足，明确说明
3. 提供实用的建议和指导
4. 必要时建议寻求专业医疗帮助
5. 语言通俗易懂，避免过于专业的术语
```
